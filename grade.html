Grade API Documentation

Base URL: http://34.121.64.128:5000/api/grades

1. Create Grade

Endpoint: /

Method: POST

Description: Creates a new grade with specified details.

Request Body:


{
  "name": "string",
  "order": 1
}

Success: 201 Created 


{
  "success": true,
  "message": "Grade was created successfully",
  "grade": {
    "id": 3409279,
    "name": "Grade Name",
    "order": 1
  }
}

Failure: 400 Bad Request 


{
  "success": false,
  "message": "Invalid grade details"
}

Possible Causes:

  • Required fields are missing or incorrect.
  • Invalid format for name or order.
  • Validation & Length:
    • name: Required, max length 255 characters.
    • order: Required, must be an integer.

2. Retrieve All Grades

Endpoint: /

Method: GET

Description: Retrieves a list of all grades.

Response:


[
  {
    "id": 3409279,
    "name": "Grade Name",
    "order": 1
  }
]

Success: 200 OK


{
  "success": true,
  "grades": [
    {
      "id": 3409279,
      "name": "Grade Name 1",
      "order": 1
    },
    {
      "id": 3409280,
      "name": "Grade Name 2",
      "order": 2
    }
  ]
}

Failure: 500 Internal Server Error 


{
  "success": false,
  "message": "Failed to retrieve grades"
}

Possible Causes:

  • Database connection issues.
  • Server error or misconfiguration.

3. Retrieve Grade by ID

Endpoint: /:id

Method: GET

Description: Retrieves details of a specific grade by ID.

Response:


{
  "id": 3409279,
  "name": "Grade Name",
  "order": 1
}

Success: 200 OK


{
  "success": true,
  "grade": {
    "id": 3409279,
    "name": "Grade Name",
    "order": 1
  }
}

Failure: 404 Not Found 


{
  "success": false,
  "message": "Grade not found"
}

Possible Causes:

  • Grade ID does not exist.
  • Database or server issues.

4. Update Grade

Endpoint: /:id

Method: POST

Description: Updates an existing grade with the specified details.

Request Body:


{
  "name": "string",
  "order": 1
}

Success: 200 OK


{
  "success": true,
  "message": "Grade updated successfully",
  "grade": {
    "id": 3409279,
    "name": "Updated Grade Name",
    "order": 2
  }
}

Failure: 404 Not Found 


{
  "success": false,
  "message": "Grade not found"
}

Possible Causes:

  • Grade ID does not exist.
  • Invalid update data.

5. Delete Grade

Endpoint: /:id

Method: DELETE

Description: Deletes a grade by ID.

Success: 200 OK


{
  "success": true,
  "message": "Grade deleted successfully"
}

Failure: 404 Not Found 


{
  "success": false,
  "message": "Grade to be deleted not found"
}

Possible Causes:

  • Grade ID does not exist.
  • Database or server issues.
