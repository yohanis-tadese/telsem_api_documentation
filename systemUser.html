System User API Documentation

Base URL: http://34.121.64.128:5000/api/system_users

1. Create System User

Endpoint: /

Method: POST

Description: Creates a new system user with specified details.

Request Body:


{
  "full_name": "John Doe",
  "email_address": "john.doe@example.com",
  "phone_number": "1234567890",
  "group_id": 1,
  "last_logged_in": "2024-09-16T00:00:00.000Z"
}

Success: 201 Created 


{
  "success": true,
  "message": "SystemUser was created successfully",
  "data": {
    "id": 1,
    "full_name": "John Doe",
    "email_address": "john.doe@example.com",
    "phone_number": "1234567890",
    "group_id": 1,
    "last_logged_in": "2024-09-16T00:00:00.000Z"
  }
}

Failure: 400 Bad Request 


{
  "success": false,
  "message": "Email address already in use"
}

Possible Causes:

  • The email address might already be registered.
  • The request body might be missing required fields.

Description: Retrieves a list of all system users.

[

    "phone_number": "1234567890",
    "group_id": 1,

{

3. Get System User by ID

Endpoint: /:id

Method: GET

Description: Retrieves a system user by its ID.

Response:


{
  "id": 1,
  "full_name": "John Doe",
  "email_address": "john.doe@example.com",
  "phone_number": "1234567890",
  "group_id": 1,
  "last_logged_in": "2024-09-16T00:00:00.000Z"
}

Success: 200 OK


{
  "id": 1,
  "full_name": "John Doe",
  "email_address": "john.doe@example.com",
  "phone_number": "1234567890",
  "group_id": 1,
  "last_logged_in": "2024-09-16T00:00:00.000Z"
}

Failure: 404 Not Found 


{
  "success": false,
  "message": "System user not found!"
}

  • The specified system user ID might not exist.

5. Delete System User

Endpoint: /:id

Method: DELETE

Description: Deletes a system user by its ID.

Success: 204 No Content

Failure: 404 Not Found 


{
  "success": false,
  "message": "System user not found!"
}

Possible Causes:

  • The specified system user ID might not exist.
