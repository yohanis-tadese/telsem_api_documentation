Course API Documentation

Base URL: http://34.121.64.128:5000/api/courses

1. Create Course

Endpoint: /

Method: POST

Description: Creates a new course with specified details.

Request Body:


{
  "title": "string",
  "description": "string",
  "thumbnail": "string",
  "grade_subject_id": 1
}

Success: 201 Created 


{
  "success": true,
  "message": "Course was created successfully",
  "data": {
    "id": 3409279,
    "title": "Course Title",
    "description": "Course Description",
    "thumbnail": "thumbnail.jpg",
    "grade_subject_id": 1,
    "createdAt": "YYYY-MM-DDTHH:MM:SSZ",
    "updatedAt": "YYYY-MM-DDTHH:MM:SSZ"
  }
}

Failure: 400 Bad Request 


{
  "success": false,
  "message": "Invalid course details"
}

Possible Causes:

  • Required fields are missing or incorrect.
  • Invalid format for title, description, or thumbnail.
  • Validation & Length:
    • title: Required, max length 45 characters.
    • description: Required, max length 100 characters.
    • thumbnail: Required, max length 20 characters.
    • grade_subject_id: Required, must be a valid integer.

3. Retrieve Course by ID

Endpoint: /:id

Method: GET

Description: Retrieves details of a specific course by ID.

Response:


{
  "id": 3409279,
  "title": "Course Title",
  "description": "Course Description",
  "thumbnail": "thumbnail.jpg",
  "grade_subject_id": 1,
  "createdAt": "YYYY-MM-DDTHH:MM:SSZ",
  "updatedAt": "YYYY-MM-DDTHH:MM:SSZ"
}

Success: 200 OK


{
  "success": true,
  "message": "Course retrieved successfully",
  "data": {
    "id": 3409279,
    "title": "Course Title",
    "description": "Course Description",
    "thumbnail": "thumbnail.jpg",
    "grade_subject_id": 1,
    "createdAt": "2024-09-14T00:00:00Z",
    "updatedAt": "2024-09-14T00:00:00Z"
  }
}

Failure: 404 Not Found 


{
  "success": false,
  "message": "Course not found"
}

  • Course ID does not exist.
  • Database or server issues.

5. Delete Course by ID

Endpoint: /:id

Method: DELETE

Description: Deletes a specific course by ID.

Success: 204 No Content 


{
  "success": true,
  "message": "Course deleted successfully"
}

Failure: 404 Not Found 


{
  "success": false,
  "message": "Course not found"
}

Possible Causes:

  • Course ID does not exist.
  • Database or server issues.
