Lookup Category API Documentation

Base URL: http://34.121.64.128:5000/api/lookup-categories

1. Create Lookup Category

Endpoint: /

Method: POST

Description: Creates a new lookup category with specified details.

Request Body:


{
  "key": "string",
  "lookup_name": "string",
  "description": "string"
}

Success: 201 Created 


{
  "success": true,
  "message": "Lookup category created successfully",
  "data": {
    "id": 1,
    "key": "example-key",
    "lookup_name": "Example Name",
    "description": "Example Description"
  }
}

Failure: 400 Bad Request 


{
  "success": false,
  "message": "Lookup category already exists."
}

Possible Causes:

  • Lookup category with the same key and lookupName already exists.

2. Retrieve All Lookup Categories

Endpoint: /

Method: GET

Description: Retrieves a list of all lookup categories.

Response:


[
  {
    "id": 1,
    "key": "example-key",
    "lookup_name": "Example Name",
    "description": "Example Description"
  }
]

Success: 200 OK


{
  "success": true,
  "message": "Lookup categories retrieved successfully",
  "data": [
    {
      "id": 1,
      "key": "example-key",
      "lookup_name": "Example Name",
      "description": "Example Description"
    }
  ]
}

Failure: 404 Not Found 


{
  "success": false,
  "message": "No lookup categories found"
}

Possible Causes:

  • No lookup categories exist in the database.

3. Retrieve Lookup Categories by Key

Endpoint: /:key

Method: GET

Description: Retrieves lookup categories matching the specified key.

Response:


[
  {
    "id": 1,
    "key": "example-key",
    "lookup_name": "Example Name",
    "description": "Example Description"
  }
]

Success: 200 OK


{
  "success": true,
  "message": "Lookup categories retrieved successfully",
  "data": [
    {
      "id": 1,
      "key": "example-key",
      "lookup_name": "Example Name",
      "description": "Example Description"
    }
  ]
}

Failure: 404 Not Found 


{
  "success": false,
  "message": "No lookup categories found with key example-key"
}

Possible Causes:

  • No lookup categories found for the specified key.

4. Update Lookup Category by ID

Endpoint: /:id

Method: PATCH

Description: Updates details of a specific lookup category by ID.

Request Body:


{
  "key": "string",
  "lookup_name": "string",
  "description": "string"
}

Success: 200 OK


{
  "success": true,
  "message": "Lookup category updated successfully",
  "data": {
    "id": 1,
    "key": "updated-key",
    "lookup_name": "Updated Name",
    "description": "Updated Description"
  }
}

Failure: 400 Bad Request 


{
  "success": false,
  "message": "Invalid lookup category details"
}

Possible Causes:

  • Required fields are missing or incorrect.

5. Delete Lookup Category by ID

Endpoint: /:id

Method: DELETE

Description: Deletes a specific lookup category by ID.

Success: 204 No Content 


{
  "success": true,
  "message": "Lookup category deleted successfully"
}

Failure: 404 Not Found 


{
  "success": false,
  "message": "Lookup category not found"
}

Possible Causes:

  • Lookup category ID does not exist.
