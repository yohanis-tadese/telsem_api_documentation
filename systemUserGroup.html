System User Group API Documentation

Base URL: http://34.121.64.128:5000/api/system-user-groups

1. Create System User Group

Endpoint: /

Method: POST

Description: Creates a new system user group.

Request Body:


{
  "name": "string",
  "description": "string",
  "roles": [
    {
      "canView": true,
      "canViewDetail": true,
      "canAdd": true,
      "canUpdate": true,
      "canDelete": false,
      "page": "string",
      "title": "string"
    }
  ]
}

Response:


{
  "id": 1,
  "name": "Admin",
  "description": "Administrative Group",
  "roles": [
    {
      "canView": true,
      "canViewDetail": true,
      "canAdd": true,
      "canUpdate": true,
      "canDelete": false,
      "page": "Dashboard",
      "title": "Dashboard"
    }
  ]
}

Success: 201 Created 


{
  "success": true,
  "message": "System user group was created successfully",
  "data": {
    "id": 1,
    "name": "Admin",
    "description": "Administrative Group",
    "roles": [
      {
        "canView": true,
        "canViewDetail": true,
        "canAdd": true,
        "canUpdate": true,
        "canDelete": false,
        "page": "Dashboard",
        "title": "Dashboard"
      }
    ]
  }
}

Failure: 400 Bad Request 


{
  "success": false,
  "message": "Validation error or invalid request"
}

Possible Causes:

  • Missing required fields in the request body.
  • Validation errors.
